About Project :

1- Instant-Simultaneous data collection from production site (eg machine stops).

2- Organizes data and stores it in a central database (integrated back and forth with ERP)

3- Planning and engineering also ensures that all data is accessible over the network, including critical data from other information systems stored in the database, such as the office.

4- It takes work orders from ERP and organizes them and transfers them to production units. At the same time, the case reacts with concurrent detailed planning to increase productivity and quality.


Detail :

1- VB.Net and C# WinForm software languages ​​were used in the project.

2- The data received from the production equipment were instantly recorded in the database on the central server.

3- By making PLC connections with HMI devices, data entry was provided.

4- Instant data is received and sent from all equipment used in the production site.

5- All data is displayed live on graphic screens.

6- All data is saved in the database.

7- Data is analyzed with reporting tools.

8- Automatic notification is made according to the event situations.
